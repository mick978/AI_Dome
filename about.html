关于 AgentAtlas

AgentAtlas 是一个精选 AI Agent 目录,收录国内外主流的 Agent 框架、平台与工具,帮助开发者快速找到合适的 AI Agent 解决方案。

收录范围

  • 框架 (framework):LangChain、AutoGen、CrewAI 等开发框架
  • 平台 (platform):Dify、FastGPT、Coze 等低代码平台
  • 工具 (tool):browser-use、agenta 等专项工具
  • 基础设施 (infrastructure):Higress 等网关层方案
  • 协议 (protocol):MCP 等标准化协议
  • 产品 (product):面向终端用户的 AI Agent 产品

❌ 不收录纯 LLM(如 GPT、Claude API),只收录 Agent 相关的框架与产品。

数据来源

所有条目均经过人工审核,优先收录:

  • GitHub Stars 较高的开源项目
  • 有官方网站和清晰文档的成熟产品
  • 国内外兼顾(目标 1:1 比例)
  • 持续维护中(最近一年内有更新)

技术栈

  • 纯静态:HTML + CSS + JavaScript
  • 零依赖:无需 npm install,无需构建
  • 双击 index.html 即可打开
  • 响应式设计:支持手机/平板/桌面
  • 暗色主题:护眼 + 优雅

部署

站点托管在 GitHub Pages,通过 git push 自动部署。域名暂用 GitHub Pages 子域名,后续可绑定独立域名。

贡献与反馈

欢迎推荐新的 Agent 项目或反馈问题。可通过 GitHub Issues 提交。

协议

本目录的代码采用 MIT 协议。收录的项目各自保留原始协议。
